📋 Google Sheet Setup — ONE TIME
  1. Sheet kholo: Click karo
  2. Row 1 mein exactly ye headers daalo:
    page  |  subject  |  sem  |  type  |  link  |  desc  |  status
  3. File → Share → Publish to web
  4. Sheet1 select karo → Format: CSV → Publish
  5. Done! ✅ Website automatically sheet se data load karegi
